ConsoleColour.java - enum of console colours used to add colour to user interface 
FileSearcher.java - finds file user has inputtted using directory user has inputted
ImageProcess.java - applies filter, exposure bias and multiplication factor, outputs image to users desired location 
Kernel.java - has all usable filters
Menu.java - Asks user to input image, pathway to image, choose a filter, change exposure bias and multiplication factor
            Asks user for output details (file name, location)
Runner.java - Runs the programme, calls Menu
